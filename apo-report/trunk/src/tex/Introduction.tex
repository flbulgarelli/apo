\section{Introducción}
Pure object aspect es un conjunto de herramientas utiles para poder aspectear
objetos facilmente. Centrado puramente en conceptos de POO.
Es una Abstraccion de Javassist.

Una de las implementaciones es Pure Object Transaction, Investiga un particular
enfoque de diseño para hacer frente a los sistemas transaccionales en un entorno orientado a objetos.
